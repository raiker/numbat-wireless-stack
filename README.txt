Import this project into your omnetpp workspace. To do this
follow the steps in this document:

http://titania.ctie.monash.edu.au/netperf/netperf-omnetpp-hypercube.pdf


Modify omnetpp.ini with your radio numbers. Using the example we have in
the omnetpp.ini, then run two instances of SPR in two separate shell windows:

Shell Window 1: /SPR -c a117
Shell Window 1: /SPR -c a113

You will see that radios sending packets to each other.
